# Data

This folder contains the basketball injury biomechanical dataset used for the injury recurrence prediction project.

The dataset includes player-level injury, rehabilitation, and biomechanical variables such as:
- Age
- Height
- Weight
- Position
- Injury type
- Injury severity
- Rehabilitation program
- Rehabilitation time
- Injury recurrence
- Knee angle
- Jump height
- Ankle flexion
- Speed
- Reaction time

The response variable is `Injury_Recurrence`.
